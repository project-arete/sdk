## Development Environment

Install Rust, then:

```shell
$ cargo add arete-sdk
```

## Testing

```shell
$ cargo test
```

## Integration Testing

```shell
$ TEST_ARETE_URL=wss://dashboard.test.cns.dev cargo test
```
